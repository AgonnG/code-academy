Course Description

60 hours/10 weeks

What you will gain from the training Introduction to Object Oriented Programming (OOP) and the basics of OOP Main pillars in OOP Part I and Part II Elaboration of some design principles Coding of real application using lessons from OOP PHP & MySQL

To complete the training successfully, students are expected to have good knowledge in BootStrap, jQuery, PHP and MySQL

Upon completion of this training the student will be able to write web applications applying the principles of object-oriented programming.
